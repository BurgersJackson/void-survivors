Map editor

Place premade shapes on a grid. The dashed circle is the spawn safety zone—keep it clear. The game loads maps/map1.json, map2.json, … in order (unlimited). Map name is stored in the file and shown in-game.

Spawns & visuals

Optional. visualSet picks one of six enemy color palettes (0–5). Grunt types and time-based waves are global (same on every map). Migrating flocks are VS-style crossing swarms.

Tilemap

The game streams tiles around the player. Tiling sets how the authored map repeats: fixed bounds, horizontal strip, vertical strip, or all directions. World items spawn in-game at a random spot 5–10 tiles from the marker (using this map’s tile size), then clamped to the play area.

Map size (world)
